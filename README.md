# Golf Charity System

A full-stack web application designed to manage a golf-based charity platform where users can participate, donate, and support charity events. Admin can manage users, events, prize pools, and run lucky draws.

User Side:
- User Registration & Login
- View Events / Competitions
- Participate in Golf Charity Events
- Check Prize Pool & Results
- Track Participation Status

Admin Side:
- Admin Login Panel
- Create & Manage Events
- Manage Users
- Run Lucky Draw / Selection System
- View Donations & Participation Data
- Declare Winners

Tech Stack:
- Frontend: React.js, Bootstrap, HTML, CSS
- Backend: Node.js, Express.js
- Database: MongoDB / PostgreSQL (as used)
- Tools: Git, GitHub, VS Code

Install dependecies:
-npm install

Run Backend:
-node server
Run Forntend:
-npm run dev
